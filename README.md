# Upcoming.orgThis is a repo for managing machine readable artifacts for the [Upcoming.org APIs](http://upcoming.yahoo.com/services/api/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Upcoming.org APIs](http://upcoming.yahoo.com/services/api/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://upcoming.yahoo.com/services/api/)- [Developer](http://upcoming.yahoo.com/services/api/)- [Twitter](https://twitter.com/Yahoo)The [APIs.json](https://github.com/api-evangelist/upcoming-org/blob/master/apis.json) index provides links to all the relevant links for the Upcoming.org APIs, including the OpenAPI and Postman if they are present.
